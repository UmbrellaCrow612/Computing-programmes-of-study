- Other types of errors and exeptions 


| Error Type        | Description                   | Example                                |
| ----------------- | ----------------------------- | -------------------------------------- |
| **Syntax error**  | Code breaks grammar rules     | Missing colon, bracket                 |
| **Runtime error** | Occurs during execution       | Division by zero, index out of range   |
| **Logic error**   | Code runs but wrong output    | Wrong operator (`and` instead of `or`) |
| **NameError**     | Variable/function not defined | `found = false` (should be `False`)    |
| **ValueError**    | Wrong value type              | `int("abc")`                           |
| **TypeError**     | Operation on wrong type       | `print(int("hello"))`                  |
